A propos de cette page

Retour à l'accueil

Le premier usage de cette page est avant tout de me permettre d'avoir un espace agréable (à l'utilisation) pour diffuser des articles sur des sujets qui m'intéressent (on peut spéculer sur le type de contenu de cette page... je ne pense pas me tromper en prédisant un maximum de documents en rapport à la programmation!)

Fonctionnement général de l'application

L'idée derrière cette application est de pouvoir décrire facilement une entrée dans la liste des publications. Une Makefile s'occupe d'appeler les générateurs correspondants aux composantes à générer.

Code source et technologies

La conversion du Markdown vers PDF et HTML est assurée par Pandoc, les différents générateurs sont implémentés avec l'excellent langage OCaml (que je recommande à toute personne désireuse d'utiliser un langage moderne et agréable). Le Javascript est généré avec Js_of_ocaml, une manière saine de faire du Javascript !

  • Code source du projet github.com/xvw/xvw.github.io

N'hésitez pas à contribuer au code (ou ... à fixer l'orthographe :) ), les données qui sont utilisées pour générer des pages se trouvent dans les répertoires : raw, raw/articles et tpl. (Désolé pour l'indentation, j'ai sérieusement échoué dans la customisation de mon .emacs)
